# Balaji Trading Company Website

A modern, responsive website for Balaji Trading Company - Authorized JK Tyres Dealer.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Complete Information**: Detailed company information, products, and services
- **Contact Form**: Interactive contact form for customer inquiries
- **Easy Navigation**: Smooth scrolling navigation with mobile-friendly menu

## Files Included

- `index.html` - Main HTML file
- `styles.css` - All styling and responsive design
- `script.js` - Interactive functionality and animations
- `README.md` - This file

## How to Use

1. Open `index.html` in any modern web browser
2. No server or installation required - works directly from the file system
3. To customize:
   - Update contact information in the Contact section
   - Add your actual address, phone numbers, and email
   - Modify company details in the About section
   - Update the year in the footer if needed

## Customization Guide

### Update Contact Information

Edit the Contact section in `index.html`:
- Replace `[Your Store Address Here]` with your actual address
- Replace `[Your Phone Number]` with your phone numbers
- Update email addresses if needed
- Modify business hours if different

### Update Company Information

Edit the About section in `index.html`:
- Modify the company description
- Update the year in the footer (replace `[Year]`)

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Notes

- The contact form currently shows an alert on submission. To make it functional, you'll need to connect it to a backend service or email service.
- All placeholder text marked with `[Your...]` should be replaced with actual information.
- The website uses Google Fonts (Poppins) - requires internet connection for fonts to load.

## Color Scheme

- Primary Color: Red (#d32f2f)
- Secondary Color: Orange (#ff6f00)
- Dark: #212121
- Light Background: #f5f5f5

You can customize these colors in the `:root` section of `styles.css`.
